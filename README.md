# 시고르

## 시스템

- 세로형 화면
- 시골 맵이 존재, 집문서 보유자들은 맵에 집을 지을 수 있음
- 화폐는 인절미 사용, 인절미 체인을 기반으로 동작
- 메타버스 내 아이템들은 NFT(721)형태로 존재하며, 인절미 체인에 기록됨
- 누구든지 NFT를 만들 수 있으며 꾸미기 가능
- NFT에는 코드를 기록할 수 있음

## 참새 NFT의 역할

참새 NFT 전용 캐릭터를 사용할 수 있음

## 집문서 NFT의 역할

참새 NFT 홀더들에게 1:1로 에어드롭된 집문서 NFT로는, 시고르 내에 집을 지을 수
있고, 하우징 가능
